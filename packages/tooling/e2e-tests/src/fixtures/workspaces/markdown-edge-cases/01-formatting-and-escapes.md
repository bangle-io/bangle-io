# Formatting and escapes

**bold** __also bold__ *italic* _also italic_ ~~struck~~ and ***combined***.

Intraword underscores should remain literal: `snake_case` and alpha_beta_gamma.

Escaped markers: \*not italic\*, \_not italic\_, \# not a heading, and
\[not a link\](https://example.com).

An inline span containing a backtick uses a longer delimiter: ``one ` tick``.

Entities and punctuation: &amp; &copy; < > “smart quotes” — em dash … ellipsis.
